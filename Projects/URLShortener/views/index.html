URL Shortener Microservice: API

Overview:

  1. POST a URL to [project_url]/api/shorturl/new and receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":1}
  2. Valid URL format http(s)://www.example.com(/more/routes)
  3. the shortened URL redirects to the original link.

POST URL here:

Example Usage:

[this_project_url]/api/shorturl/1

Will Redirect to:

https://www.google.com
